# Node.js and MongoDB CRUD App

A simple RESTful web service using Node.js, Express, and MongoDB for managing people records. Features include Create, Read, Update, and Delete (CRUD) operations with EJS templates for server-side rendering.

---

## Features
- **View People**: Displays a table with all records in the database.
- **Add Person**: Form to create a new person with fields for name, age, gender, and mobile number.
- **Edit Person**: Form to edit and update an existing person.
- **Delete Person**: Confirmation page to delete a specific person from the database.

---

## Tech Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose ODM
- **Frontend**: EJS Templates
- **Styling**: Basic CSS

---

## Installation

### Prerequisites
1. Install [Node.js](https://nodejs.org/).
2. Install [MongoDB](https://www.mongodb.com/docs/manual/installation/) and ensure it is running.
3. Clone this repository:
   ```bash
   git clone https://github.com/your-username/node-mongo-crud-app.git
   cd node-mongo-crud-app

---

## steps to run

1. Install dependencies: npm install
2. Start MongoDB: brew services start mongodb/brew/mongodb-community@6.0
3. Start the server: node server.js
4. Access the application in your browser at: http://localhost:3000/person
